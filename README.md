# Social_Network_API
## Description:
This project is to built an API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list.
This uses Express.js for routing, a MongoDB database, and the Mongoose ODM as well as the Express.js and Mongoose packages.
